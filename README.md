# 로스트아크 파티 매칭 도구

원정대 캐릭터를 레이드별로 자동으로 4인 파티/공격대로 편성해주는 웹 도구입니다. 입장 가능 여부는 아이템 레벨로 체크하고, 파티 밸런스는 **인게임 전투력** 기준으로 맞춰줍니다.

## 주요 기능

- **원정대 자동 불러오기**: 캐릭터명 하나로 원정대 전체 가져오기 (Lostark Open API)
- **전투력 기반 파티 편성**: `/armories/characters/{name}` API로 캐릭터별 인게임 전투력을 받아와 snake-draft로 평균 전투력을 균등 분배 (로아와, 클로아와 동일한 값)
- **여러 공격대 자동 생성**: 같은 레이드 구간에 캐릭이 많으면 공격대 1, 공격대 2, ... 자동으로 분할
- **4인 / 8인 레이드 구분**: 세르카·지평의 성당은 파티 1개, 종막·카제로스는 공격대(4인 파티 × 2개)
- **원정대 중복 방지**: 한 파티에 같은 원정대(=같은 계정) 캐릭은 1명만 (로아 실제 입장 제약 반영)
- **레이드 선택 체크박스**: 원하는 레이드만 켜서 파티 편성, 카테고리별 전체/해제
- **드래그 앤 드롭**: 파티 간 캐릭 이동 가능, 이동 시 평균 자동 재계산
- **중복 배정 표시**: 한 캐릭이 여러 레이드에 걸쳐 있으면 `×2` `×3` 배지로 표시
- **레벨 구간 수정**: 패치로 입장 레벨 바뀌면 UI에서 직접 편집
- **커스텀 레이드 추가**: 신규 레이드나 임의 구간 추가 가능
- **로컬 저장**: 모든 데이터는 브라우저 localStorage에만 저장 (API 키 포함, 서버 전송 없음)

## 시작하기

1. 로스트아크 Open API 키 발급: https://developer-lostark.game.onstove.com
   - STOVE 로그인 → `GET ACCESS TO LOSTARK API` → `CREATE A NEW CLIENT`
   - CLIENT NAME만 영문으로 입력, 나머지 공백, 체크박스 체크 후 Submit
   - 생성된 Client에서 **API KEY COPY**
2. 웹페이지 좌측 "API 키" 입력란에 붙여넣고 **저장**
3. "원정대 등록"에서 아무 캐릭터명 입력 → **원정대 추가** 클릭
   - 원정대 목록 + 각 캐릭의 전투력을 순차적으로 불러옴 (캐릭터 수만큼 API 호출, 보통 15~20초)
4. 왼쪽에서 파티 짤 레이드 체크박스 선택
5. 오른쪽에 자동으로 파티 편성 결과 표시

## GitHub Pages 배포

1. GitHub에 저장소 생성 (예: `lostark-party`)
2. 파일들 업로드 (Public 저장소여야 Pages 무료)
3. **Settings → Pages → Source**: Deploy from a branch, `main` 브랜치 `/ (root)`
4. 1~2분 뒤 `https://<아이디>.github.io/lostark-party/` 로 접속

**GitHub Desktop으로 쉽게 업데이트하기 (권장)**:
1. https://desktop.github.com 설치 후 로그인
2. 본인 저장소 Clone → 로컬 폴더에 복사됨
3. 앞으로 파일 수정 후 GitHub Desktop에서 **Commit to main → Push origin** 버튼 2번
4. 1~2분 뒤 Pages에 자동 반영

## 파티 매칭 알고리즘

**입장 필터링**: 각 레이드의 `[minLevel, maxLevel)` 반개구간에 해당하는 캐릭만 후보.
예: 세르카 노말(1710~1730)은 1710.00 ~ 1729.99 아이템 레벨 캐릭만.

**공격대 반복 생성**: 후보로 공격대 1개를 만들고, 남은 후보로 다시 공격대 2개, 3개... 더 이상 파티 하나도 못 만들 때까지 반복.

각 공격대 내부:
1. 서포터를 전투력 높은 순으로 각 파티(4인 단위)에 1명씩 배치
2. 딜러를 전투력 내림차순 snake-draft로 배분 (파티 간 평균 전투력 균등화)
3. 딜러 배정 시 **같은 원정대 캐릭이 이미 파티에 있으면 건너뜀** (로아 실제 입장 제약)
4. 서포터 부족한 파티엔 딜러 4명으로 구성 후 "⚠ 서포터 부족" 표시
5. 원정대 다양성 부족으로 4명을 못 채우면 "⚠ 인원 미달" 표시

**전투력 못 받아온 캐릭**:
- armories API가 실패하거나 시크릿 설정된 캐릭은 전투력이 0으로 옴
- 이 경우 임시로 `템레벨 × 2`로 폴백 (대략적 비례)
- 칩에 작은 `?` 배지 표시

## 한계 및 참고사항

- **같은 캐릭 다중 레이드 배정**: 1725 캐릭이 세하N + 카제로스 4막N 둘 다 가능한 경우, 각 레이드에 독립 할당됩니다. 칩에 `×2` 배지가 뜨고, 마우스 올리면 어느 레이드들에 들어있는지 확인 가능. 본인이 드래그로 조정하세요.
- **입장 레벨 패치 대응**: 2026년 4월 기준. 패치로 변경 시 각 레이드 "레벨수정" 버튼으로 직접 편집.
- **API Rate Limit**: 분당 100회. 원정대 추가 시 캐릭 수만큼 호출되니, 원정대 5개 이상을 동시에 새로고침하면 429 오류 가능. 150ms 간격으로 호출하며, 429 발생 시 10초 대기 후 1회 재시도합니다.
- **전투력 API**: `/armories/characters/{name}?filters=profiles` 응답의 `ArmoryProfile.CombatPower` 값 사용. 이는 로아와/클로아가 쓰는 값과 동일 (인게임 전투력 그대로).
