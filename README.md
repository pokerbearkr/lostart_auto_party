# 로스트아크 파티 매칭 도구

원정대 캐릭터를 레이드별 아이템 레벨 구간에 맞춰 자동으로 4인 파티로 편성하고, 드래그로 자유롭게 조정할 수 있는 웹 도구입니다.

## 주요 기능

- **원정대 자동 불러오기**: 캐릭터명 하나만 입력하면 해당 원정대의 모든 캐릭터를 불러옵니다 (Lostark Open API 사용)
- **여러 원정대 동시 관리**: 여러 계정의 원정대를 한 화면에서 관리
- **자동 파티 매칭**: 각 레이드의 입장 레벨 구간에 해당하는 캐릭터만 자동으로 4인 파티(서포터 1 + 딜러 3)로 편성
- **평균 템렙 균형**: snake-draft 알고리즘으로 각 파티의 평균 아이템 레벨을 최대한 균등하게 분배
- **드래그 앤 드롭**: 완성된 파티에서 캐릭터를 드래그로 다른 파티로 이동 가능 (이동 시 평균 자동 재계산)
- **서포터 부족 표시**: 서포터가 부족하면 해당 파티에 경고 표시
- **프리셋 + 커스텀 레이드**: 발탄부터 세하 더퍼스트까지 주요 레이드 내장 + 사용자 커스텀 레이드 추가 가능
- **레벨 구간 수정**: 패치로 입장 레벨이 바뀌면 UI에서 직접 수정 가능
- **로컬 저장**: 모든 데이터는 브라우저 localStorage에만 저장 (서버 전송 없음)

## 시작하기 (로컬 테스트)

1. 이 저장소를 클론하거나 ZIP으로 받습니다.
2. `index.html`을 브라우저로 열면 바로 동작합니다.
   - (Chrome의 경우 `file://` 에서도 정상 동작)
3. 로스트아크 Open API 키를 발급받습니다: https://developer-lostark.game.onstove.com
   - 회원가입 → 로스트아크 선택 → API 키 발급 (무료)
4. 웹페이지 좌측 "API 키" 입력란에 붙여넣고 저장.
5. "원정대 등록"에서 원정대의 아무 캐릭터명이나 입력 → "원정대 추가" 클릭.
6. 레벨 구간에 맞는 레이드가 자동으로 표시되며 파티가 편성됩니다.

## GitHub Pages 배포

1. 본 프로젝트를 GitHub 저장소에 push 합니다 (예: `lostark-party-matcher`).
2. 저장소 설정 → **Settings** → **Pages** 이동.
3. **Source**: "Deploy from a branch" 선택.
4. **Branch**: `main` (또는 `master`) + `/root` (또는 `/ (root)`) 선택 후 **Save**.
5. 1-2분 후 `https://<username>.github.io/<repo-name>/` 에서 접속 가능합니다.

### 주의: HTTPS 환경 필수

로스트아크 API는 HTTPS를 요구합니다. GitHub Pages는 기본적으로 HTTPS를 제공하므로 배포 후 정상 동작합니다. 로컬에서 `file://`로 열었을 때 CORS 문제가 생기면, 간단한 로컬 서버로 테스트하세요:

```bash
# Python
python -m http.server 8000
# Node.js
npx serve .
```

## 파티 매칭 알고리즘

각 레이드의 `[minLevel, maxLevel)` 반개구간에 해당하는 캐릭터만 후보로 추립니다. 예를 들어 세하 노말이 1710, 하드가 1730이면, 세하 노말 파티에는 **1710.00 ~ 1729.99** 캐릭터만 들어갑니다.

1. 후보를 **서포터/딜러**로 분리
2. 생성 가능한 파티 수 = `min(서포터 수, floor(딜러 수 / 3))`
3. 각 파티에 서포터 1명 배치
4. 딜러를 레벨 내림차순 정렬 후 **snake-draft**로 배분하여 파티 간 평균이 고르도록 분배
5. 서포터는 충분한데 딜러가 모자라거나, 남은 딜러가 4명 이상이면 "서포터 부족 파티(딜러 4명)" 으로 별도 표시
6. 여전히 남는 캐릭터는 "남은 캐릭터" 영역에 표시 → 드래그로 파티에 추가 가능

## 한계 및 참고사항

- 같은 캐릭터가 여러 레이드 구간에 걸쳐 있으면(예: 1725 캐릭이 세하N/카제로스 4막N 둘 다 가능) 각 레이드에 독립적으로 할당됩니다. 이때 칩에 `×2`, `×3` 같은 주황색 배지가 표시되어 "이 캐릭이 여러 레이드에 동시 배정됐음"을 한눈에 확인할 수 있습니다. 칩에 마우스를 올리면 정확히 어느 레이드들에 들어가 있는지 보여주며, 드래그로 제외하면 배지가 실시간 업데이트됩니다.
- 입장 레벨은 2026년 4월 기준이며, 패치로 달라질 수 있습니다. 레이드 우측 "레벨수정" 버튼으로 구간을 조정할 수 있습니다.
- API는 분당 호출 제한이 있습니다 (100회/분). 여러 원정대를 한 번에 새로고침할 때 429 오류가 나면 잠시 후 재시도하세요.
- 본 도구는 Smilegate RPG와 무관한 팬메이드 프로젝트입니다.
